# Player Sprite

## Installation

Bitte lade das Player-Sprite-Bild hier hoch:

**Pfad:** `/workspaces/gatefall/public/assets/sprites/player.png`

Das Bild sollte:
- Ein Charakter-Sprite sein (idealerweise stehend, Vollkörper)
- Transparenten Hintergrund haben (PNG)
- Ca. 400-600px Höhe
- Im Hochformat sein

## Aktuelles Bild

Das hochgeladene Bild wird automatisch für alle Player-Sprites verwendet (Wächter, Jäger, Magier).

Das Bild wird mit einem bläulichen Glow-Effekt und abgerundeten Ecken angezeigt.
